# Password Generator
## Implemented in flask react

### Create virtual flask env
`python -m venv <env_name>`

### Activate  environtment
`flaskenv\Scripts\activate`

### Install flask and other lib
`pip install -r requirements.txt`

### run the app
`python app.py`
